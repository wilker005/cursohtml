O que é HTML?

(VOLTAR AO TOPO)

Criado por Tim Berners-Lee, HyperText Markup Language (Linguagem de Marcação de Hipertexto) ou HTML, é a estrutura que compõe a maioria dos sites. Nele pode ter textos, imagens, vídeos,etc.

O que são Tags?

(VOLTAR AO TOPO)

As Tags são códigos que montam a estrutura do HTML como por exemplo, adicionar imagens, parágrafos, títulos, divisões quebras de linha, etc.

Tags de Formatação de Texto

(VOLTAR AO TOPO)

As Tags de Formatação de Texto servem para mudar a aparência do texto escrito. As principais Tags são:

H1 ao H6

Eles representam 6 difderentes níveis de título, com tamanhos diferentes sendo o H1 o maior e o H6 o menor.


EXEMPLOS:


Essa frase foi escrita com H1;


Essa frase foi escrita com H2;


Essa frase foi escrita com H3;


Essa frase foi escrita com H4;


Essa frase foi escrita com H5;

Essa frase foi escrita com H6.

Tags OL, UL e LI

(VOLTAR AO TOPO)

As Tags OL e UL são usadas para criar listas. OL significa Ordered List(Lista Ordenada), ao criar a lista com OL, a lista aparecerá em ordem numérica 1,2,3... Já UL significa Unordered List(Lista Não Ordenada), ao criar a lista com UL, a lista não aparecerá em ordem numérica.

A Tag LI é a lista dos itens, ela é usada para representar cada item presente na lista.

EXEMPLO

LISTA ORDENADA

  1. ITEM 1
  2. ITEM 2
  3. ITEM 3

LISTA NÃO ORDENADA

  • ITEM 1
  • ITEM 2
  • ITEM 3
